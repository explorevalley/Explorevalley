Add tours and hotels using the Telegram admin bot:

- Send `/addtour` to create a tour (follow the prompts).
- Send `/addhotel` to create a hotel (follow the prompts).

Photos uploaded in the bot are stored under `public/uploads/tours` and `public/uploads/hotels`.

Ensure `TELEGRAM_BOT_TOKEN` and `ADMIN_CHAT_IDS` are set in `server/.env`.
